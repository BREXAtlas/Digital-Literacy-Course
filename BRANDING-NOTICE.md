# Branding Notice

Ram Ready Digital Literacy is an **independent, open-source pilot** built for review and possible future
coordination with Angelo State University. It is:

- **Not** an official Angelo State University course, policy, credential, website, or AI-use policy.
- **Not** connected to Angelo State single sign-on.
- Built using an ASU-inspired color direction (blue #245397, gold #F0C33B) and "Ram Fam" themes of belonging,
  community, integrity, growth, and inclusion, referencing:
  - [Core Values](https://www.angelo.edu/about-asu/core-values/)
  - [Blue and Gold](https://www.angelo.edu/about-asu/traditions/blue-and-gold.php)
  - [Roscoe and Bella](https://www.angelo.edu/about-asu/traditions/roscoe-and-bella.php)
- **Not** using any official Angelo State University seal, logo, photograph, or copyrighted mascot artwork.
  Roscoe and Bella appear only as original, text-based interface guide characters with original interface
  treatment and original dialogue — their dialogue is never represented as official university speech.

## Course policy notice

This course does not define Angelo State's official artificial-intelligence or academic-integrity policy.
Learners are repeatedly instructed to follow the current syllabus, assignment directions, instructor guidance,
and official university policy. Course rules may differ by class and can change over time.

## Repository lineage notice

This repository (`BREXAtlas/Digital-Literacy-Course`, no trailing hyphen) is a separate project from
`BREXAtlas/Financial-Literacy-Course` (the interface/design model) and `BREXAtlas/Digital-Literacy-Course-`
(the trailing-hyphen curriculum source). No Supabase configuration, GitHub Pages configuration, or deployment
environment was copied from either reference repository. See `docs/FINANCIAL_SHELL_PARITY.md`.

If Angelo State University wishes to formally adopt, endorse, or fork this project, that requires separate
coordination outside this repository. Until then, please do not represent this project as officially
affiliated with or endorsed by Angelo State University.
